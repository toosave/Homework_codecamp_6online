1.1 CSS syntax ประกอบด้วยอะไรบ้าง

CSS syntax ประกอบด้วย Selector และ Declaration ใน Declaration ประกอบด้วย CSS Property และ CSS Value

Selector Declaration
HTML Tag CSS Property CSS Value

1.2 การเขียน CSS มีกี่แบบอะไรบ้าง

การเขียน CSS มี 3 แบบ
1. External CSS
2. Internal CSS
3. Inline CSS

1.3 selector มีกี่แบบอะไรบ้าง

selector มี 5 แบบ
1. Simple selectors (ใช้ selector ตามชื่อelements , ชื่อid หรือชื่อclass)
2. Combinator selectors (ใช้ selector ตามความสัมพันธ์แต่ละแบบของ elements)
3. Pseudo-class selectors (select elements based on a certain state)
4. Pseudo-elements selectors (select and style a part of an element)
5. Attribute selectors (select elements based on an attribute or attribute value)

1.4 combinator มีกี่แบบอะไรบ้าง

combinator มี 4 แบบ
1. descendant selector(space)
2. child selector(>)
3. adjacent sibling selector(+)
4. general sibling selector(~)
