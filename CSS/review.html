1.1 CSS syntax ประกอบด้วยอะไรบ้าง

CSS syntax ประกอบด้วย Selector และ Declaration ใน Declaration ประกอบด้วย CSS Property และ CSS Value

1.2 การเขียน CSS มีกี่แบบอะไรบ้าง

การเขียน CSS มี 3 แบบ
1. External CSS
2. Internal CSS
3. Inline CSS

1.3 selector มีกี่แบบอะไรบ้าง

selector มี 5 แบบ
1. Simple selectors (ใช้ selector ตามชื่อelements , ชื่อid หรือชื่อclass)
2. Combinator selectors (ใช้ selector ตามความสัมพันธ์ของแต่ละ elements)
3. Pseudo-class selectors (ใช้ selector ตามสถานะของแต่ละ elements)
4. Pseudo-elements selectors (ใช้ selector เพื่อกำหนดส่วนต่างๆของ elements)
5. Attribute selectors (ใช้ selector เลือกเฉพาะที่มี attributes ที่ระบุ)

1.4 combinator มีกี่แบบอะไรบ้าง

combinator มี 4 แบบ
1. descendant selector(space)
2. child selector(>)
3. adjacent sibling selector(+)
4. general sibling selector(~)

4.2 grid และ flex แตกต่างกันอย่างไร

No.flex layoutgrid layout
1. มีการกำหนดทิศทาง ของข้อมูลภายในคอนเทนเนอร์ ไปในทิศทางเดียวกัน เช่นแนวนอน(row) หรือแนวตั้ง(column) สามารถกำหนดทิศทาง ของข้อมูลภายในคอนเทนเนอร์ ไปได้พร้อมกันทั้ง แนวนอน(row) และแนวตั้ง(column)
2. ให้ความสำคัญกับการจัดเนื้อหาในคอนเทนเนอร์ เป็นลำดับแรก ให้ความสำคัญกับการจัดแบ่งพื้นที่ ที่ชัดเจนของคอนเทนเนอร์ เป็นลำดับแรก
3. flex เหมาะสมกับโครงสร้างเว็บ ที่ส่วนใหญ่เป็นแบบเส้นตรง(แนวนอนหรือแนวตั้ง) หรือโครงสร้างขนาดเล็ก ในขณะที่grid เหมาะสมกับโครงสร้างเว็บขนาดใหญ่ ที่มีการกำหนดโครงสร้างทั้งแนวตั้ง , แนวนอน หรือทับซ้อนกัน

4.3 justify and align แตกต่างกันอย่างไร

alignเป็นการจัดคอนเทนเนอร์ ตามแกน column เช่นให้ชิดด้านบน กลาง หรือล่าง
justifyเป็นการจัดคอนเทนเนอร์ ตามแกน row เช่นให้ชิดด้านซ้าย กลาง หรือขวา

5.1 position คืออะไร และ อะไรบ้าง

ValueWhat is the Position
static static เป็นค่าเริ่มต้นของทุก element จะแสดงตำแหน่งที่คงที่ ตามลำดับในเอกสาร การใช้คำสั่ง top หรือleft จะไม่มีผล
relative element อยู่ในตำแหน่งที่อ้างอิงกับตำแหน่งของ element หลัก การใช้คำสั่งเช่น left: 20px element จะห่างจากตำแหน่งซ้ายของ element หลัก 20พิกเซล
fixed element อยู่ในตำแหน่งที่อ้างอิงกกับหน้าต่างเบราว์เซอร์ การใช้คำสั่งเช่น left: 20px element จะห่างจากตำแหน่งซ้ายของหน้าต่างเบราว์เซอร์ 20พิกเซล
absolute element อยู่ในตำแหน่งที่อ้างอิงกับตำแหน่งของ element หลักลำดับแรกสุด การใช้คำสั่งเช่น left: 20px element จะห่างจากตำแหน่งซ้ายของ element หลักลำดับแรกสุด 20พิกเซล
sticky การรวม relative กับ fix ไว้ด้วยกัน โดยการกำหนด relative เพิ่มว่าจะให้ไป fix ที่ top หรือ bottom ทำให้เราเห็น block นั้นเลื่อน และถูกตรึงไว้เมื่อเราเลื่อน scroll เมาส์

ValueWhat is the z-index
