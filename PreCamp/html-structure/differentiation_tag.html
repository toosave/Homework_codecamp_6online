2. ให้หาความเหมือนและความแตกต่างระหว่าง 2 tag ที่กำหนดให้
<hr>&<br> , <ul>&<ol> , <select>&<checkbox>&<redio button> , <div>&<span> , <script>&<style>

Differentiation Tag

Tag เหมือนกัน ต่างกัน
<hr>&<br> ใช้สำหรับขึ้นบรรทัดใหม่ <br>จะเป็นการขึ้นบรรทัดใหม่โดยไม่มีเส้นคั่นหน้า
<hr>จะเป็นการขึ้นบรรทัดใหม่และมีเส้นคั่นหน้าด้วย
<ul>&<ol> ใช้สำหรับสร้างรายการเรียงลำดับ <ul>เป็นรายการเรียงลำดับ โดยใช้สัญลักษณ์อยู่หน้ารายการ
<ol>เป็นรายการเรียงลำดับ โดยใช้ตัวเลขหรือตัวอักษรอยู่หน้ารายการ
<select>&<checkbox>&<radio button> ใช้สำหรับสร้างรายการตัวเลือก <select> คือรายการตัวเลือกแบบ drop-down
<checkbox> คือรายการตัวเลือกแบบ มีช่องสี่เหลี่ยมให้เลือก สามารถเลือกได้หลายรายการ
<radio button> คือรายการตัวเลือกแบบ มีช่องวงกลมให้เลือก เลือกได้หนึ่งรายการ
<div>&<span> ใช้กำหนดส่วน หรือแบ่งส่วนเอกสาร HTML <div> ใช้กำหนดส่วน หรือแบ่งส่วนเอกสารขนาดใหญ่กว่า
<span> ใช้กำหนดส่วน หรือแบ่งส่วนเอกสารที่เป็นส่วนย่อยอยู่ภายใน
<script>&<style> ใช้เพิ่มองค์ประกอบคำสั่งจากภายนอก เพื่อเข้ามาจัดการในเอกสาร HTML <script> ใช้เพิ่มองค์ประกอบคำสั่งด้วย javascript
<style> ใช้เพิ่มองค์ประกอบคำสั่งด้วย CSS
