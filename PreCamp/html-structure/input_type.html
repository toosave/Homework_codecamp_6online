HTML structure ☕

3. ให้หาว่า input type มีอะไรบ้างแต่ละ type ต่างกันอย่างไร

Input Type

Input Type การใช้งาน
button เป็น input แบบปุ่มกด
checkbox เป็น input แบบมีช่องตัวเลือกสี่เหลื่ยม
color เป็นinput แบบมีตัวเลือกสี และแสดงสีที่เลือก
date เป็นinput แบบมีตัวเลือกวันที่ และแสดงวันที่ที่เลือก
datetime-local เป็นinput แบบมีปุ่มกดขึ้นลง และ drop down ให้เลือกเดือน, วัน, ปี, ชั่วโมง, นาที และ am pm
email เป็นinput แบบมีช่องรับข้อความตัวหนังสือในรูปแบบ "ตัวอักษร@ตัวอักษร" ถ้าใส่ข้อมูลไม่ครบ ขอบช่องรับข้อความจะเป็นสีแดงเตือน เพื่อให้ใส่ขอมูลให้ครบ
file เป็นinput แบบปุ่มกด "Browse" เพื่อเลือก file upload
hidden เป็นinput แบบที่ทำให้ไม่มีการแสดงผล หรือถูกซ่อนไว้
image เป็นinput แบบที่สามารถแสดงผลด้วยรูปภาพได้
month เป็นinput แบบมีปุ่มกดตัวเลือกเดือนและปี
number เป็นinput แบบมีช่องรับข้อความตัวเลข ถ้าใส่ข้อมูลตัวอักษร ขอบช่องรับข้อความจะเป็นสีแดงเตือน เพื่อให้ใส่ขอมูลตัวเลข
password เป็นinput แบบมีช่องรับข้อความรหัสผ่าน ขณะพิมพ์จะแสดง เป็นสัญลักษณ์ขึ้นมาแทนข้อความรหัสผ่าน
radio เป็นinput แบบปุ่มกดวงกลม ถ้ามีหลายรายการที่มีค่า name เดียวกัน จะกดเลือกได้รายการใดรายการหนึ่ง
range เป็นinput แบบปุ่มแถบเลื่อน สามารถเลื่อนไปมาได้ ระหว่างค่า min และค่า max ที่ตั้งไว้
reset เป็นinput แบบปุ่มกด ถ้ากดปุ่มนี้จะเป็นการลบค่าใน input อื่นๆที่อยู่ใน form ทั้งหมด
search เป็นinput แบบมีช่องรับข้อความตัวอักษร,ตัวเลข และอักขรพิเศษบางตัว
submit เป็นinput แบบปุ่มกด ถ้ากดปุ่มนี้ข้อมูลใน form จะถูกส่งไปยังเว็บที่เขียนไว้ที่ attribute action
tel เป็นinput แบบมีช่องรับข้อความ ถ้ามีการใช้กับ attribute pattern กับ required ด้วย regex pattern จะสามารถกำหนดรูปแบบข้อความที่จะรับได้
text เป็นinput แบบเป็นช่องรับข้อความทั่วไป
time เป็นinput แบบป็นช่องรับตัวเลข และมีปุ่มกดขึ้นลงให้กดเลือก ชั่วโมง นาที AM PM
url เป็นinput แบบเป็นช่องรับข้อความรูปแบบ URL "http://ข้อความ"
week ป็นinput แบบป็นช่องรับตัวเลข และมีปุ่มกดขึ้นลงให้กดเลือก สัปดาห์(1-53) ปี
