HTML structure ☕

1. ให้หาว่า attribute ของ tag ต่อไปนี้มี อะไรบ้างและใช้งานอย่างไร
<h>, <p>, <div>, <img>, <a>, <ul>, <ol>, <teble>, <form>, <button>

Attribute

Tag Attribute การใช้งาน
<h> align จัดตำแหน่ง left, center, right
<p> align จัดตำแหน่ง left, center, right
<div> align จัดตำแหน่ง left, center, right
<img> alt ระบุข้อความสำรองสำหรับรูปภาพ
crossorigin อนุญาตให้ใช้รูปภาพจากไซต์ของบุคคลที่สามที่อนุญาตให้ใช้ cross-origin ใช้กับ Canvas
height ระบุความสูงของรูปภาพ
ismap ระบุรูปภาพ เป็นรูปภาพฝั่งเซิร์ฟเวอร์
longdesc ระบุ URL โดยละเอียดของรูปภาพในคำอธิบาย
referrerpolicy ระบุผู้อ้างอิงที่จะใช้เวลาดึงภาพ
sizes กำหนดขนาดของรูปภาพสำหรับ page layout
src ระบุ path ไปยังรูปภาพ
srcset แสดง URL ของรูปภาพนั้นๆเพื่อนำไปใช้ในจุดประสงค์ต่างๆ
usemap ระบุชื่อmapในรูปภาพที่มี link ซ่อนอยู่ และมีการกำหนดขนาด map ภายใน ด้วย tag map และ tag area
width ระบุความกว้างของรูปภาพ
<a> download ระบุว่าเป้าหมายจะถูกดาวน์โหลด เมื่อผู้ใช้คลิกที่ hyperlink
href ระบุ URL ของหน้าที่ลิงก์ที่จะไป
hreflang ระบุภาษาของเอกสารใน link
media ระบุสื่อหรืออุปกรณ์ ที่ลิงค์ไว้กับเอกสาร
ping ระบุ link แจ้งเตือนใน ping เพื่อใช้แจ้งให้เจ้าของเว็บไซต์ทราบหากมีการคลิกลิงค์
referrerpolicy ระบุผู้อ้างอิงที่จะส่ง
rel ระบุความสัมพันธ์ระหว่างเอกสารปัจจุบันและเอกสารใน link
target ระบุตำแหน่งที่จะเปิดเอกสารใน link
type ระบุประเภท media type ของเอกสารใน link
<ul> compact ระบุให้ลิสแสดงผลออกมาเป็นขนาดเล็ก
type ตั้งค่า bullet style เป็น circle, disc หรือsquare
<ol> reversed ระบุว่าลำดับรายการจะลดลง เช่น (9,8,7...)
start ระบุค่าเริ่มต้นของรายการ
type ระบุชนิดของเครื่องหมายที่จะใช้ในรายการ เช่น (1, A, a, I, i)
<table> align จัดตำแหน่ง left, center, right
bgcolor ระบุสีพื้นหลังของตาราง
border กำหนดขนาดพิกเซลของกรอบตาราง
cellpadding กำหนดช่องว่างระหว่างเนื้อหาของเซลล์และเส้นขอบของเซลล์
cellspacing กำหนดขนาดช่องว่างระหว่างเซลล์
frame กำหนดด้านข้างของเฟรมที่ล้อมรอบตารางที่ต้องแสดง
rules กำหนดกฎ การแสดงเส้นตาราง
summary กำหนดข้อความที่สรุปเนื้อหาของตาราง
width กำหนดความกว้างของตาราง
<form> accept-charset ระบุการเข้ารหัสอักขระที่จะใช้สำหรับการส่งแบบฟอร์ม
action ระบุตำแหน่งเอกสารที่จะส่งข้อมูลแบบฟอร์ม เมื่อมีการส่งแบบฟอร์ม
autocomplete ระบุการเปิดหรือปิด การเติมข้อความอัตโนมัติ
enctype ระบุวิธีการเข้ารหัสข้อมูลแบบฟอร์มเมื่อส่งไปยังเซิร์ฟเวอร์ (เฉพาะ method="post")
method ระบุ HTTP method ที่จะใช้ เมื่อส่งแบบฟอร์มข้อมูล
name ระบุชื่อของแบบฟอร์ม
novalidate ระบุว่าไม่ต้องตรวจสอบความถูกต้องของฟอร์มเมื่อส่ง
rel ระบุความสัมพันธ์ระหว่างตำแหน่งส่งแบบฟอร์ม กับเอกสารปัจจุบัน
target ระบุตำแหน่งที่จะแสดงการตอบสนอง หลังจากส่งแบบฟอร์ม
<button> autofocus ระบุให้มีการปรับโฟกัสโดยอัตโนมัติ เมื่อกดปุ่ม เช่นเมื่อใช้กับการแสดง alert หน้าเอกสารจะปรับมาชัดที่ alert
disabled ระบุให้ปุ่มปิดการใช้งาน
form ระบุว่าปุ่มใช้นี้กับ form id ชื่ออะไร ใช้เมื่อปุ่มอยู่นอก tag form
formaction ระบุตำแหน่งเอกสารที่จะส่งข้อมูลแบบฟอร์ม เมื่อจะส่งแบบฟอร์มไปยังเอกสารที่ไม่ได้ระบุใน atrribute action ใน tag form ใช้กับ type = "submit" เท่านั้น
formenctype ระบุวิธีเข้ารหัสข้อมูลในแบบฟอร์มก่อนส่งไปยังเซิร์ฟเวอร์ ใช้กับ type = "submit" เท่านั้น
formmethod ระบุวิธีการส่ง HTTP method ถ้าใน tag form ไม่ได้ระบุไว้จะใช้ method GET ในการส่ง ใช้กับ type = "submit" เท่านั้น
formnovalidate ระบุว่าไม่ต้องตรวจสอบความถูกต้องของข้อมูลแบบฟอร์มเมื่อส่ง ใช้กับ type = "submit" เท่านั้น
formtarget ระบุตำแหน่งที่จะแสดงการตอบสนองหลังจากส่งแบบฟอร์ม ใช้กับ type = "submit" เท่านั้น
name ระบุชื่อสำหรับปุ่ม
type ระบุประเภทของปุ่ม
value ระบุค่าเริ่มต้นสำหรับปุ่ม
